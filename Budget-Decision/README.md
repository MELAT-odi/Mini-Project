# Personal Budget Decision Tool

A simple web app that helps users decide whether they can afford to buy an item based on their budget and savings.

## Project Description

This project analyzes a user's financial situation and determines if a purchase is affordable.  
It validates user input and provides clear financial feedback.

## Features

- Interactive financial prompts
- Input validation with loops
- Budget calculation logic
- Real-time result display on the webpage
- Clean conditional decision system

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
